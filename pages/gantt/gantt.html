Como Personalizar Este Gráfico de Gantt

  • Para ajustar a duração de uma tarefa, modifique a largura e a posição das barras .task-period no CSS.
  • Altere as cores das tarefas modificando as classes .task-1 até .task-5 no CSS.
  • Para adicionar mais tarefas, copie a estrutura HTML de uma .task-row existente e atualize o conteúdo.
  • Para modificar o número de dias, ajuste o número de colunas no grid e as células de dias.
